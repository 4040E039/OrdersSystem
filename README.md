# About OrdersSystem

## 目的
**簡化公司內部訂餐流程，讓訂餐更快速且方便**

## 使用工具 / 技術
Laravel Jetstream、Tailwind、Vue3

## 如何安裝
1. composer install.
2. npm install.
3. npm run dev.
4. copy .env.example .env.
5. php artisan key:generate.
6. php artisan storage:link.
7. php artisan migrate.
